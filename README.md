### 개발 역량

#### 구현 내용
- Spring Framework의 MVC 패턴 적용, JPA 기반 도메인과 엔티티 연관관계 설계
- WebSocket, STOMP 기반 실시간 통신 기능 구현 경험
- Spring Security와 JWT를 활용한 인증·인가 구조 구현 경험
- JUnit 기반 테스트 코드 작성 및 기능 검증 경험
- Flutter FE를 맡아, REST API 엔드포인트를 기준으로 요청·응답 구조 파악, 인증 토큰과 데이터 화면에 연결
- Jenkins surge 슬롯을 활용한 점진적 Rolling 배포 구성 경험
- Nginx 기반 리버스 프록시 및 멀티 슬롯 로드밸런싱 구성 경험
- Cloudflare Tunnel 기반 홈서버 외부 노출 및 Zero Trust 환경 구성 경험

#### 개선 경험
- Pagination Fetch Join 트러블 슈팅 상세 조회와 목록 조회 DTO를 분리하여 해결, 응답시간 99.6% 개선
- EXPLAIN으로 옵티마이저의 실행 계획을 분석하고, 조회 조건과 정렬 기준에 맞는 복합 인덱스를 적용
- Polling 방식의 메시지 기능을 WebSocket과 Redis Pub/Sub 기반 실시간 전파 구조로 개선
- 서비스 중단을 최소화하기 위해 로드밸런싱 기반의 SpringBoot 인스턴스를 2개 구성하고 점진적 롤링 배포 구현
- WAS 스케일 아웃 환경에서 세션 불일치 문제를 해결하기 위해 JWT 기반 인증을 적용하고, Redis를 활용해 서버 측 무효화 구조 구현
- 비관적 락으로 크레딧 거래 DB 동시성 문제 해결 및 Lock 순서를 통일해 DeadLock 문제 방지
- 동시성 제어 중 데이터 비갱신 문제, 트랜잭션 격리 수준 및 Fetch Join 조정하여 해결
